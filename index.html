My works, projects & contributions

Hyper-spectral image segmentation of head and neck squamous cell carcinoma

Developed and optimized a Wavelet CNN on an HPC to segment hyper-spectral images, training it to leverage spatial & spectral features for multi-class segmentation of head & neck cancers (100+ patients). Achieved 0.74 IoU on unseen testing data.

View

3D Unet-based cardiac segmentation of neonatal mouse

Developed and tuned a 3D Unet to segment neonatal mouse hearts from a 1000+ image stack, preprocessing and augmenting the cardiac light-sheet data. Achieved 0.84 IoU in accurately segmenting three distinct cardiac structure.

View

Emotion Recognition Tool

Developed machine learning system to classify human emotions in real-time video. Trained convolutional neural networks on FER-2013 dataset with 35,000 facial images. Achieved 82% accuracy in detecting 7 emotions using GPU-optimized TensorFlow/Keras framework.

View

Pima Indian Diabetes Prediction

Developed random forest classifier and performed exploratory data analysis then engineered features from dataset of 768 patients. Employed perceptron model optimized through hyperparameter tuning and achieved 89% accuracy on unseen test data.

View

Tech Mahindra Project - Truck Fuel Efficiency Prediction

Developed neural network model to predict truck fuel efficiency using real-world telemetry data. Extracted throttle input readings from sensor streams to correlate with MPG measurements. Employed regression algorithms achieving 85% accuracy in MPG estimates.
